Zmienne

Zmienne w JavaScript są używane do przechowywania danych. Można je deklarować za pomocą var, let lub const. 

let zmienna = 'Hello, World!';

Różnice między var, let i const:

  • var: Deklaruje zmienną globalnie lub lokalnie w funkcji, może być redeklarowana i ma zakres funkcji.
  • let: Deklaruje zmienną lokalnie w bloku, nie może być redeklarowana w tym samym zakresie, ma zakres bloku.
  • const: Deklaruje stałą, której wartość nie może być zmieniona, ma zakres bloku.

Funkcje

Funkcje to bloki kodu, które wykonują określone zadania. Można je definiować i wywoływać w dowolnym miejscu w kodzie.

Funkcja deklarowana

function przykladFunkcji() {
    console.log('To jest funkcja');
}

Funkcja wyrażeniowa

const przykladFunkcji = function() {
    console.log('To jest funkcja');
};

Funkcja strzałkowa

const przykladFunkcji = () => {
    console.log('To jest funkcja');
};

Parametry i argumenty

Parametry to zmienne zadeklarowane w definicji funkcji, a argumenty to wartości przekazywane do funkcji podczas jej wywoływania. 

function dodaj(x, y) {
    return x + y;
}

console.log(dodaj(2, 3)); // 5

W powyższym przykładzie x i y to parametry, a 2 i 3 to argumenty.

Domyślne wartości parametrów

function dodaj(x, y = 10) {
    return x + y;
}

console.log(dodaj(5)); // 15

Jeśli argument y nie zostanie przekazany, przyjmie wartość domyślną 10.

Rest parameters

function suma(...args) {
    let sum = 0;
    for (let arg of args) sum += arg;
    return sum;
}

console.log(suma(1, 2, 3, 4)); // 10

Rest parameters pozwalają na przekazanie dowolnej liczby argumentów jako tablicy.

Pętle

Pętle pozwalają na wielokrotne wykonywanie tego samego bloku kodu. Najczęściej używane pętle to for, while i do-while.

Pętla for

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Pętla while

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Pętla do-while

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Instrukcja if

Instrukcja if pozwala na wykonywanie kodu tylko wtedy, gdy spełniony jest określony warunek. 

if (warunek) {
    console.log('Warunek jest spełniony');
} else {
    console.log('Warunek nie jest spełniony');
}

Instrukcja switch

Instrukcja switch pozwala na wybór jednego z wielu bloków kodu do wykonania, w zależności od wartości wyrażenia. 

switch (wyrazenie) {
    case 'wartosc1':
        console.log('Wartość 1');
        break;
    case 'wartosc2':
        console.log('Wartość 2');
        break;
    default:
        console.log('Inna wartość');
}

Manipulacja DOM

JavaScript pozwala na dynamiczną manipulację elementami DOM, co umożliwia tworzenie interaktywnych stron internetowych.

Wybieranie elementów DOM

const element = document.querySelector('selector');

Metoda querySelector zwraca pierwszy element, który pasuje do podanego selektora.

Zmiana zawartości elementów

element.textContent = 'Nowa zawartość';

Właściwość textContent pozwala na zmianę tekstu wewnątrz elementu.

Zmiana atrybutów elementów

element.setAttribute('atrybut', 'wartość');

Metoda setAttribute pozwala na ustawienie wartości atrybutu elementu.

Zmiana stylów elementów

element.style.color = 'red';

Właściwość style pozwala na zmianę stylów CSS elementu.

Tworzenie nowych elementów

const nowyElement = document.createElement('div');
document.body.appendChild(nowyElement);

Metoda createElement tworzy nowy element, a appendChild dodaje go do DOM.

EventListener

EventListener pozwala na reagowanie na różne zdarzenia, takie jak kliknięcia, ruchy myszy, wciśnięcia klawiszy itp. 

const button = document.querySelector('button');
button.addEventListener('click', (event) => {
    console.log('Przycisk został kliknięty!');
});

Metoda addEventListener dodaje nasłuchiwacz zdarzeń do elementu.
